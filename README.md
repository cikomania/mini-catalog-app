# Mini Catalog App (Flutter)

A Flutter mini catalog app with product listing, GridView layout, cart system, login flow, and navigation.  
This project was developed as part of a Flutter training program.

##  Features

- Login screen (UI authentication simulation)
- Product listing with GridView
- Product detail screen
- Add / remove products from cart
- Cart screen with dynamic updates
- Home screen banner integration
- Basic search bar UI (non-functional)
- Multi-screen navigation
- API integration for product data

## Tech Stack

- Flutter
- Dart
- HTTP package
- Material Design

## API

Product data is fetched from:  
https://wantapi.com/products.php

## Flutter Version

Flutter SDK: 3.41.6 (stable channel)

## How to Run

1. Download or Clone the Repository  
   - Click the "Code" button on GitHub and select **Download ZIP**,  
     or clone using:

     ```bash
     git clone https://github.com/cikomania/mini-catalog-app.git
     ```

2. Open the Project  
   - Open the project in **Visual Studio Code** or Android Studio  
   - Make sure Flutter SDK is installed and an emulator or physical device is connected before running the project

3. Install Dependencies  
   - Run the following command in terminal:

     ```bash
     flutter pub get
     ```

4. Run the Application  
   - Connect an emulator or physical device  
   - Then run:

     ```bash
     flutter run
     ```
